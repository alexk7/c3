Value
Abstract entity.
Type
Set of values.
Datum
Information that can be manipulated by the computer.
Data Type
Set of data that can be produced by the computer with a common set of operations. A data type can be used to establish a correspondance between data and values. In this case, there is a type which is the set of values represented by the datatype. It can also be used to contain platform artifacts like memory addresses.
Object
Physical entity that can store a datum of a specific datatype.
Object Type
Set of data that the object can store.
State
The datum currently stored in an object.
